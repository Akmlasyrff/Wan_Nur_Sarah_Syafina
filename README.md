# WhisperBox - Anonymous Confession System

A secure, anonymous space to share your thoughts and secrets.

## Features
- **Anonymous Submissions**: Share confessions without logging in.
- **Glassmorphism Design**: Modern, beautiful UI with dark mode aesthetics.
- **Responsive**: Optimised for both desktop and mobile devices.
- **Real-time Updates**: See new confessions instantly without refreshing (optimistic UI).

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   node server.js
   ```

3. Open your browser:
   [http://localhost:3000](http://localhost:3000)

## Technology Stack
- **Frontend**: HTML5, CSS3 (Glassmorphism), Vanilla JavaScript
- **Backend**: Node.js, Express
- **Data Storage**: JSON file (local persistence)
